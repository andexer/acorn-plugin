# Acorn Plugin Skeleton 🌰 🚀

La base definitiva para construir plugins de WordPress con el poder de Laravel.
acorn-plugin es un "boilerplate" de grado profesional que integra Laravel Acorn directamente en el flujo de desarrollo de plugins de WordPress. Proporciona a los "artesanos web" herramientas modernas como el Contenedor de Servicios (IoC), Query Builder, Blade Templating y soporte para CLI, eliminando el código repetitivo y permitiendo un desarrollo más rápido, organizado y testeable.
⚡ Instalación en un solo paso
Genera un plugin base completo y listo para producción con un solo comando de Composer:

composer create-project andexer/acorn-plugin <nombre-de-tu-plugin>

✨ Características Principales

- Arquitectura Acorn: Aprovecha los componentes de Laravel (Config, Cache, Container, Logging) dentro de WordPress.
- Zero Boilerplate: Estructura de carpetas limpia y lista para empezar a programar lógica de negocio de inmediato.
- WP-CLI Ready: Genera comandos personalizados de WordPress utilizando la sintaxis de Laravel.
- Blade Engine: Olvida el PHP mezclado con HTML; utiliza componentes y vistas elegantes.
- Auto-loading: Configuración PSR-4 integrada para gestionar tus clases sin esfuerzo.

🛠 Requisitos

- PHP >= 8.3
- Composer
- Acorn instalado en tu entorno de WordPress (o cargado vía composer).
